# Momo .run安装包

在 OpenWrt 上使用 sing-box 进行透明代理。

## 环境要求

- OpenWrt >= 24.10
- Linux Kernel >= 5.13
- firewall4

## 功能

- 透明代理 (Redirect/TPROXY/TUN, IPv4 和/或 IPv6)
- 访问控制
- 配置文件编辑器
- 定时重启

## 安装

## 方法一:命令行 
```
wget -O momo.run <momo.run地址>
sh momo.sh
```
